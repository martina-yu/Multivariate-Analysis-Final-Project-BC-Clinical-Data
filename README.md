# Multivariate-Analysis-Final-Project-BC-Clinical-Data



**Variables:** 

7 categorical variables and 5 continuous variables (notice that height and weight are correlated in corr ~0.5, I made BMI in later analysis). But in the beginning, there’re still 5 continuous variables.

**Summary Table**

| Name                          | Type        | Explanation                                                  |
| ----------------------------- | ----------- | ------------------------------------------------------------ |
| **USI**                       | /           | Patient ID                                                   |
| **RNA_BATCH**                 | Categorical | RNA batch number                                             |
| **TREATMENT_ARM**             | Categorical | Drug Adjuvant Endocrine Treatment or use of hormone therapy (Yes or No) |
| **MENOSPAUSAL_STATUS**        | Categorical | Menopausal status (Post or Pre)                              |
| **AGE**                       | Continuous  | Age of patient when enrolled                                 |
| **RISK_GROUP**                | Categorical | Risk level (1: low, 2, 3, 4: high)                           |
| **Nodal Status**              | Continuous  | Number of positive lymph nodes                               |
| **Tumor Size (cm)**           | Continuous  | Size of the tumor in centimeters                             |
| **WT (kg)**                   | Continuous  | Weight of the patient in kilograms                           |
| **HT (cm)**                   | Continuous  | Height of the patient in centimeters                         |
| **Histologic Grade**          | Categorical | Grade of the tumor                                           |
| **Survival Status Indicator** | Categorical | Patient survival status                                      |
| **IDFS Event**                | Categorical | Invasive disease-free survival event                         |
